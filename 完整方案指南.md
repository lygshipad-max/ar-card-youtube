## 🔄 如何將 YouTube 轉成 MP4

如果你選擇**版本 3（本地影片）**，需要這些步驟：

### 方法 A：使用線上工具（最簡單）

#### 推薦網站：

1. **Y2Mate** - https://www.y2mate.com/
   ```
   1. 貼上 YouTube 網址
   2. 選擇 MP4 格式
   3. 選擇畫質（建議 720p）
   4. 下載
   ```

2. **SaveFrom** - https://savefrom.net/
   ```
   1. 貼上網址
   2. 點擊下載
   3. 選擇 MP4 格式
   ```

3. **9xBuddy** - https://9xbuddy.org/
   ```
   支援多種影片網站
   操作簡單
   ```

#### ⚠️ 注意事項：
- 選擇 **720p** 畫質（檔案大小和品質的平衡）
- 確認檔案 < 10MB（GitHub 限制）
- 影片長度建議 < 3 分鐘

---

### 方法 B：使用軟體（更專業）

#### Windows/Mac: 4K Video Downloader
```
下載：https://www.4kdownload.com/
功能：批量下載、選擇畫質、自動轉檔
```

#### 命令列工具: yt-dlp
```bash
# 安裝
pip install yt-dlp

# 下載影片
yt-dlp -f "best[height<=720]" https://www.youtube.com/watch?v=影片ID

# 指定輸出檔名
yt-dlp -o "video.mp4" https://www.youtube.com/watch?v=影片ID
```

**Clideo** - https://clideo.com/compress-video
```
簡單易用

```

---

### 方法 2：使用 HandBrake（免費軟體）

下載：https://handbrake.fr/

**設定：**
```
Format: MP4
Video Codec: H.264
Quality: RF 23-28 (數字越大檔案越小)
Resolution: 720p
Audio: AAC, 128 kbps
```

---

### 方法 3：FFmpeg（命令列）

```bash
# 壓縮到指定大小
ffmpeg -i input.mp4 -b:v 1M -bufsize 1M output.mp4

# 調整解析度
ffmpeg -i input.mp4 -vf scale=1280:720 output.mp4

# 更積極的壓縮
ffmpeg -i input.mp4 -vcodec libx264 -crf 28 video.mp4
```

## ❓ 常見問題

### Q: 哪個版本最接近我要的效果？
**A:** ar-youtube-embedded.html - 播放器在畫面中央，不是全螢幕，可以看到 AR 元素。

### Q: 為什麼播放器不會跟著 AR 移動？
**A:** 因為技術限制。要做到跟隨需要用本地 MP4（版本 3）。

### Q: 可以同時用 YouTube 又有 3D 整合嗎？
**A:** 不行。只能二選一：
- YouTube → 2D 覆蓋層
- 3D 整合 → 本地 MP4

### Q: 影片檔案太大怎麼辦？
**A:** 使用壓縮工具（HandBrake 或 FFmpeg）降低畫質或解析度。

### Q: 可以用其他影片格式嗎？
**A:** 建議用 MP4（H.264 編碼），相容性最好。
